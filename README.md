# Sunless Skies 한국어 패치

**현재 배포 버전: v1.1.1**

Sunless Skies 한국어 패치입니다.

아직 초벌 번역 단계입니다. 자동번역 캐시를 배포 패키지에서 제외하고, 수동으로 검수한 번역만 포함합니다.

이 프로젝트는 **Eueeeeeeeek** 님이 제작·배포한 [Sunless Skies 한글 패치](https://eueeeeeeeek.kr/162)의 번역 데이터와 한글 글꼴·패치 환경을 기반으로 시작했습니다. 기존 자료를 직접 패치 설치 방식에 맞게 재구성하고, 번역 추가·용어 정리·UI 및 시스템 문자열 보완 작업을 이어가고 있습니다. 원본 한글 패치 제작자에게 감사드립니다.

처음에는 다음 순서로 프로젝트를 진행할 계획이었습니다.

1. BepInEx를 사용한 번역 적용
2. BepInEx 의존성 제거 및 직접 패치 방식으로 전환
3. 전용 설치 프로그램 제작

그러나 작업 도중 먼저 한글 패치를 제작하신 분의 자료를 발견하여, 해당 패치를 바탕으로 2단계부터 적용하는 방향으로 전환했습니다.

v1.0.0부터는 BepInEx를 설치하지 않는 직접 패치 설치기를 사용합니다. 설치기는 게임 데이터 파일을 백업한 뒤 번역 데이터와 정적 UI 패치를 적용합니다.

현재는 초반 튜토리얼, 메뉴, 선장 생성, 뉴 윈체스터, 약속의 날들, 거래 튜토리얼, 일부 바자/교역/Prospect 문구를 중심으로 번역하고 있습니다.

## 번역 현황

원 한글 패치에 포함된 번역은 직접 패치 데이터와 정적 에셋에 이미 반영되어 있습니다. 아래 수치는 전체 번역 진행률이 아니라, 누락 문구와 UI·시스템 문자열을 보완하기 위해 설치기가 추가로 인식하는 번역 사전의 현황입니다.

- 추가 번역 키: `7,276`
- 번역 사전 파일: `79`
- 번역 사전 줄: `7,270`

게임에서 추출한 원문 후보에는 중복 문장, 고유명사, 개발용 문구가 포함되고 원 패치의 직접 패치 데이터는 번역 사전 키에 포함되지 않으므로, 이를 기준으로 전체 번역률을 계산하지 않습니다. 현재는 게임을 직접 확인하며 남아 있는 영문 UI와 누락 문장을 보완하고 있습니다.

## 변경 이력

### v1.1.1 (2026-07-12)

- 번역 사전을 저장소 최상위 `translations` 폴더로 정리했습니다.
- 배포에 필요하지 않은 기존 BepInEx 런타임과 레거시 설치 자료를 제거했습니다.
- 내부 패치·검사 도구를 용도에 따라 정리하고 실험용 도구를 별도 경로로 이동했습니다.
- 직접 패치 설치기와 GitHub Release 패키징 경로를 새 저장소 구조에 맞게 갱신했습니다.

### v1.1.0 (2026-07-12)

- 기존 번역 원본과 현재 배포 사전을 대조해 패키지에 없던 번역을 추가했습니다.
- `passive speed boost`의 의미 반전과 `howling wastes`, `disturbing landmarks` 등 문맥을 벗어난 번역을 수정했습니다.
- `Wolvesey`를 `울브지`, `Wolvesey Engine Yard`를 `울브지 기관차 작업장`으로 통일했습니다.
- `High Wilderness`에 혼재하던 `거대 황야`, `하늘 황야`, `하늘 광야`, `고지 황야` 표기를 `하이 와일더니스`로 정리했습니다.
- 원문 대조 검사 범위와 반영 결과를 [`docs/source-comparison-qa-20260712.md`](docs/source-comparison-qa-20260712.md)에 기록했습니다.

### 2026-07-12 작업 내역

- 과거 런타임 자동 수집본과 현재 배포 사전을 비교해, 패키지에 없던 번역 키 98개를 별도 보완 파일로 추가했습니다.
- 날짜·상태·보상 표시, 뉴 윈체스터의 상점과 시설, 휘틀록 선장의 블랙박스 관련 초반 이벤트 등 실제 플레이 중 남아 있던 영문을 보완했습니다.
- 번역, 스크린샷 제보, 용어, 설치기, QA, 커밋·배포 작업을 각각 독립적으로 이어갈 수 있도록 작업별 문서를 정리했습니다.
- 이번 변경은 번역 사전과 작업 문서 갱신이며, 설치기 코드와 현재 배포 버전 `v1.0.2`는 변경하지 않았습니다.

### v1.0.2 (2026-07-11)

- 전체 번역 사전 7,000여 항목을 대상으로 1차 품질 검수를 진행했습니다.

- `zailor`, `zee-captain`, `incautious`, `lurid` 등 문맥을 벗어난 기계 번역과 어색한 표현을 수정했습니다.

- 메뉴, 일시정지 화면, 도움말, 날짜·상태 표시, 지역명 등 정적 UI 에셋의 미번역 문자열 40곳을 보완했습니다.
- 번역 파일의 UTF-8 깨짐, 미번역 영문 문장, 동적 토큰 누락 및 정적 UI 잔여 문자열을 자동 검사했습니다.
- 설치 패키지를 실제 게임 폴더와 캐시를 대상으로 시험하여 번역 키 7,276개와 정적 에셋 7개가 정상 인식되는 것을 확인했습니다.

자세한 검수 범위와 남은 수동 확인 사항은 [`docs/translation-qa-20260711.md`](docs/translation-qa-20260711.md)에 기록했습니다.

## 설치 방법

1. Sunless Skies를 종료합니다.
2. Release에서 `SunlessSkies-KoreanPatch.zip`을 내려받습니다.
3. 압축을 풉니다.
4. `SunlessSkiesKoreanInstaller.exe`를 실행합니다.
5. 설치가 끝나면 게임을 다시 실행합니다.

설치 프로그램이 게임 폴더를 자동으로 찾지 못하면 안내에 따라 직접 게임 폴더를 입력할 수 있습니다. 설치 전 원본 데이터 파일은 `_backup_korean_direct_patch_날짜` 폴더에 백업됩니다.

기본 Steam 설치 경로는 보통 아래와 같습니다.

```text
C:\Program Files (x86)\Steam\steamapps\common\Sunless Skies
```
