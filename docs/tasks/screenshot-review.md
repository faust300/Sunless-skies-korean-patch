# 스크린샷 제보 처리

## 목표

사용자가 보여 준 화면만 고치는 데서 끝내지 않고 같은 화면과 연결된 미번역 문자열을 함께 보완한다.

## 작업 순서

1. 화면에 보이는 영문 중 길고 고유한 구절부터 검색한다.
2. 해당 항목의 제목, 본문, 선택지, 결과, 툴팁을 함께 찾는다.
3. 같은 이벤트 ID 또는 인접한 데이터 항목을 확인한다.
4. 이미 번역된 부분과 용어·시제를 맞춰 한 묶음으로 수정한다.
5. 화면 잘림이 제보된 경우 뜻을 유지하면서 더 짧은 표현을 우선한다.

## 자주 확인할 화면

- 메인 메뉴와 일시정지 메뉴
- 튜토리얼 팝업과 조언 창
- 항구, 바자, 지역명과 상호작용 안내
- 날짜, 연료, 보급품, 경험치와 상태 툴팁
- 이벤트 제목, 본문, 선택지, 결과 및 보상 문구
- 선장 생성, 야망, 과거와 업적 화면

## 주의점

- 화면에 한국어와 영어가 섞이면 서로 다른 데이터 소스나 정적 에셋일 수 있다.
- `</`, `<`, `[dir:...]` 같은 조각이 화면에 노출되면 번역보다 태그 구조를 먼저 의심한다.
- 글꼴 문제와 미번역 문제를 구분한다. 네모 출력은 번역 사전이 아니라 글꼴 에셋 문제일 가능성이 높다.
