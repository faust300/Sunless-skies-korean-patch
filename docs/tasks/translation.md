# 일반 번역 작업

## 목표

미번역 영문과 어색한 한국어를 빠르게 수정해 로컬 작업 트리에 누적한다.

## 작업 순서

1. 원문 전체 또는 식별 가능한 일부 문장으로 `rg` 검색한다.
2. 같은 원문이 여러 파일에 있으면 실제 로드 순서와 기존 번역을 확인한다.
3. 앞뒤 이벤트, 선택지, 결과 문장을 함께 읽고 번역한다.
4. 변수, 숫자 자리표시자, 태그와 동적 토큰을 그대로 보존한다.
5. 수정한 항목 주변만 확인하고 다음 번역으로 넘어간다.

## 원칙

- 초벌 번역 단계에서는 번역량 확보를 우선하고 문학적 문체 검수는 후순위로 둔다.
- 자동번역 문장을 그대로 채택하지 않고 최소한 문맥과 고유명사를 확인한다.
- 번역 한 건마다 전체 검사, 빌드, 커밋 또는 푸시를 하지 않는다.
- 기존 한국어가 게임 데이터에 이미 적용된 경우 영어 원문 키 기반 갱신이 먹지 않을 수 있음을 유의한다.

## 파일 형식

- 번역 파일은 UTF-8을 유지한다.
- 원문과 번역의 구분 형식을 바꾸지 않는다.
- `=`, 색상 태그, 강조 태그가 구문 일부일 때는 기존 이스케이프 방식을 따른다.
