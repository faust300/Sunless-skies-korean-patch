# 작업 문서 안내

이 폴더는 긴 대화 컨텍스트를 대신하는 작업별 기록이다. 현재 요청과 관련된 문서만 읽고 작업한다.

| 문서 | 용도 |
| --- | --- |
| `translation.md` | 일반 번역 추가 및 문맥 수정 |
| `screenshot-review.md` | 게임 스크린샷에서 미번역·오역 찾기 |
| `terminology.md` | 고유명사와 반복 용어 통일 |
| `installer.md` | C# 설치기와 직접 패치 구조 변경 |
| `qa.md` | 번역 및 패키지 검증 |
| `release.md` | 커밋, 푸시, 버전, GitHub Release |

새로운 결정은 관련 문서에만 기록한다. 여러 문서에 같은 내용을 중복해서 적지 않는다.
